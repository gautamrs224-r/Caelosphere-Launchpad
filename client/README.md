# Caelosphere Launchpad — Frontend (Stage 2)

Frontend-only build using static dummy data, per the project plan (Stage 2: Complete Frontend Development).

## Run locally
```
npm install
npm run dev
```

## Build
```
npm run build
```

## Pages included
- Landing page (hero, features, how it works, testimonials, CTA, footer)
- Login / Register
- Dashboard home + Analytics
- Startup Creation Wizard (5 steps) + AI Analysis loading screen
- Projects list
- Workspace Overview
- Settings
- Placeholder empty states for History / Exports / Reports

No backend, database, or AI integration yet — all data is static (see `src/data/dummy.js`).
